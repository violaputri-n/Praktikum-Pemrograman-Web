🧠 Keterampilan & Tools

Berikut beberapa kemampuan dan tools yang saya kuasai selama menempuh pendidikan Teknik Informatika.

💻 PHP

Bahasa pemrograman backend yang saya gunakan untuk membuat sistem berbasis web dinamis.

⚙️ Laravel

Framework PHP modern untuk membangun aplikasi web dengan struktur MVC yang efisien dan rapi.

⚛️ React

Library JavaScript untuk membangun antarmuka pengguna interaktif dengan komponen yang dinamis.

🐍 Python

Digunakan untuk analisis data, AI, dan machine learning menggunakan library seperti NumPy & TensorFlow.

🗄️ MySQL

Sistem manajemen basis data yang digunakan untuk menyimpan, mengelola, dan memanipulasi data proyek.

🌐 HTML & CSS

Dasar dalam membangun struktur dan tampilan website dengan desain responsif dan modern.

📱 JavaScript

Digunakan untuk membuat interaksi dinamis dan animasi ringan pada antarmuka website.

🧩 Git & GitHub

Alat version control untuk mengelola proyek secara kolaboratif dan menyimpan versi kode dengan aman.

🎨 Figma

Tool desain antarmuka (UI/UX) untuk membuat prototipe interaktif dan desain tampilan aplikasi.

🤖 AI Tools

Menggunakan alat berbasis AI untuk membantu pembuatan kode, desain, serta analisis data proyek.
